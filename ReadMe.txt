First page:
	1. User choose to register to specialist,
	2. To check appointment by unique code,
	3. Or if it's specialist - to log in.

Client registering:
	Client can choose specialist -> date -> time -> leads to the reservation page.
	There reservation can be canceled or checked if specelist not canceled it.

Specialist login:
	Every specialist has his password (Vytautas is 101). Page shows current
	or upcoming meeting. Appointment can be started, stoped, canceled. Once
	started can not be chosen another client.

Information screen:
	Can be reach throught specialist login page with code 0000
